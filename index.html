Vectary

3 years at Vectary maintaining and replacing their cluster of microservices making up the APIs used by the entire software solution.

  • Initial maintenance of REST-like microservices in NodeJS/Typescript. Using a relational database without relations and storing binaries in AWS's expensive Elastic File System which where indexed with Elasticsearch to allow for fuzzy searches.
  • Replacement of the cluster of microservices with a GraphQL API using an improved database schema, S3 for binary storage, a system to prevent duplicated binaries and PostgreSQL's TSVector and TSQuery types for fuzzy searches. Achieving better performance at one fifth of the cost while also offering new features.

Vectary site
